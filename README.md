# PHP Extras

[![JetBrains Marketplace](https://plugins.jetbrains.com/static/versions/35649/logo.svg)](https://plugins.jetbrains.com/plugin/13757-php-extras)

<!-- Plugin description -->
<p>Adds additional extra features that are not otherwise found in other plugins to enhance development.</p>
<p>The current features added are:</p>
<ul>
    <li>Remove composer vendor include paths for attached modules from Preferences &gt; Languages &amp; Frameworks &gt; PHP &gt; Include Path</li>
    <li>Adds a quick fix for adding a module directly from the composer.json</li>
</ul>
<!-- Plugin description end -->
